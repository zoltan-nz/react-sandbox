# React TypeScript Sandbox

## npm scripts

```
$ npm run clean
$ npm run prettier
$ npm run start:dev
$ npm run lint
$ npm run build
$ npm run test:watch
$ npm run test
```

## Add bootstrap

```
$ npm install --save bootstrap reactstrap
$ npm i --save-dev @types/reactstrap
```

Add to `./src/index.tsx`:

```
import 'bootstrap/dist/css/bootstrap.css';
```
